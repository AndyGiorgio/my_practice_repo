Web Accessibility Check

User Characteristic Accsessible Design Tip
Unable to see Code all images with ALT text
Unable to perceive color Avoid using color alone to convey information
Unable to use mouse Be sure all website feautures can be accessible using the keyboard
Unable to hear Add captions to multimedia
Prone to having seizures Avoid content flashes
Easily distractable Keep the design simple
